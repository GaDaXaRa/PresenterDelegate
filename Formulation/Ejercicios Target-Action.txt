#Ejercicios Target-Action

##Ejercicio 1. Nuestro primer botón.
- Poner título al botón para el estado normal y pulsado.
- Cuando se pulse el botón, llamar al método *buttonPressed*
- Cuando se suelte el botón o se arrastre fuera del mismo llamar al método *buttonReleased*.

##Ejercicio 2. Añadiendo imágenes al botón.
- Poner la imagen *icn-beer-normal* como para el estado normal y la imagen *icn-beer-selected* para el estado pulsado. 
- Poner las imágenes de *button_background* para los estados normal y pulsado.
     
##Ejercicio 3 Poniendo texto a una etiqueta.
- Ponerle el texto que os guste
- Cambiarle el tipo de fuente a Georgia usando el método *fontWithName:* en la clase *UIFont*
- Cambiar el color de la fuente.
    
##Ejercicio 4 Juntando botones y etiquetas.
- Crear dos botones y una etiqueta de forma que cuando pulsemos el primer botón la etiqueta tenga alineación a la izquierda y cuando pulsemos el segundo la etiqueta tenga alineación a la derecha. 
    
    *Nota: puede ser interesante cambiar el color de fondo de la etiqueta para que se vea mejor la alineación*

##Ejercicio 5. UISwitch
Utilizando un switch, vamos a cambiar el *lineBreakMode* de una etiqueta. Cuando esté en apagado utilizaremos el *truncating tail* y cuando esté encendido el *truncating head*. 

##Ejercicio 6. UISlider
Usando un UISlider vamos a cambiar el tamaño de fuente de una etiqueta.

##Ejercicio 7. UISegmentedControl

 Usando un UISegmentedControl vamos cambiar entre los diferentes tipos de *lineBreakMode* de una etiqueta cuando ésta tiene múltiples líneas. Para ello deberemos poner el *numberOfLines* igual a 2, por lo menos.
